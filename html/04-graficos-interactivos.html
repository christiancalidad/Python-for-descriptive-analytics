 Bokeh¶

Bokeh es una librería para graficación desarrollada por Continuum Analytics. El tutorial de Bokeh puede consultarse en el sitio oficial de la herramienta, cuyo vínculo está suministrado en la bibiografía.

En este tutorial se presenta un gráfico en Bokeh que tiene como fin probar que la librería está correctamente instalada.

In [8]:
# El primer paso consiste en importar las librerias en Jupyter.
import numpy as np
from bokeh.plotting import figure, output_file, show
from bokeh.io import output_notebook
In [9]:
output_notebook()
Loading BokehJS ...
In [10]:
# se generan puntos aleatorios
N = 4000
x = np.random.random(size=N) * 100
y = np.random.random(size=N) * 100
radii = np.random.random(size=N) * 1.5
colors = [
    "#%02x%02x%02x" % (int(r), int(g), 150) for r, g in zip(50+2*x, 30+2*y)
]
In [11]:
# genera la figura, grafica los puntos y visualiza la gráfica
p = figure(x_range=(0,100), y_range=(0,100))
p.circle(x,y, radius=radii, fill_color=colors, fill_alpha=0.6, line_color=None)
show(p)

Seaborn¶

Seaborn by Michael Waskom.
